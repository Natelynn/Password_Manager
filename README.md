# Password Manager (MVC)

**Автор:** Дмитрий Пшенников

Консольный менеджер паролей на базе архитектуры **MVC** с применением принципов **ООП** (инкапсуляция данных, наследование и полиморфизм генераторов). Позволяет структурировано хранить аккаунты и локально сохранять их в JSON-файл.

## Функционал
- Просмотр, добавление и удаление записей.
- Настраиваемая генерация надежных паролей (длина, цифры, спецсимволы).
- Автоматическая валидация полей и обработка ошибок ввода.
- Локальное хранение данных в `passwords.json`.

## Запуск
```bash
python main.py
```

## Тест-кейсы
1. **Позитивный:** Добавление корректных данных (пункт 2) -> запись успешно выводится в таблицу (пункт 1).
2. **Негативный:** Отправка пустых полей при создании -> срабатывает валидация `[-] Ошибка валидации: Поля не могут быть пустыми!`.
3. **Граничный:** Ввод несуществующего (например, `99`) или нечислового индекса при удалении -> перехват ошибки, приложение продолжает работу.
